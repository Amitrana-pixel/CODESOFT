# 🚢 Task 1: Titanic Survival Prediction

## 📖 Overview

The objective of this project is to predict whether a passenger survived the Titanic disaster based on passenger information such as age, gender, passenger class, fare, and family details.

### Dataset

- Titanic Dataset
- Contains passenger details and survival status.

### Workflow

- Data Loading
- Data Cleaning
- Handling Missing Values
- Label Encoding
- Exploratory Data Analysis (EDA)
- Feature Selection
- Train-Test Split
- Model Training
- Model Evaluation
- Prediction

### Machine Learning Algorithms Used

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

### Evaluation Metrics

- Accuracy Score
- Confusion Matrix
- Classification Report
